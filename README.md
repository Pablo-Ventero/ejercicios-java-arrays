## ▶️ Cómo ejecutar cualquier ejercicio

**Requisitos:** tener Java instalado.
```bash
# Entrar a la carpeta del ejercicio
cd 01-array-orden-inverso

# Compilar
javac Principal.java

# Ejecutar
java Principal
```

## 📚 Conceptos cubiertos
- Arrays estáticos y recorrido con índices
- `ArrayList` y métodos: `add()`, `get()`, `set()`, `size()`
- Recorrido inverso de colecciones
- Búsqueda de máximo y mínimo
- Conversión de bases numéricas
- Búsqueda y reemplazo de elementos
